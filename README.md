# Padaria Pão de Ouro — Análise de Vendas

## Sobre o Projeto
Projeto de análise de dados desenvolvido como parte do meu portfólio 
de transição de carreira para a área de dados.

O objetivo foi analisar o desempenho de vendas de uma padaria fictícia 
durante a primeira semana de janeiro de 2024, respondendo perguntas 
reais de negócio que um proprietário precisaria para tomar decisões.

## Ferramentas Utilizadas
- **Google Sheets** limpeza e organização dos dados
- **Power BI** criação do dashboard interativo
